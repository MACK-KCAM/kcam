# kcam

immersive photo albums

## Building and running
- Open two terminals 
  - In the first 
    - run `npm i`
    - Build the server (with `npm run server`)
  - In the second
    - run `npm i`
    - Build the client side (with `npm run start`)
- Open browser to http://localhost:8081/

## Tech Stack
### Frontend Stack
- JavaScript
- React360 (with React Native)

### Backend Stack
- Express
- MongoDB
